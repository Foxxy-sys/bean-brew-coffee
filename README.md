# ☕ Bean & Brew Coffee

A modern and responsive coffee shop company profile website built with **HTML5**, **CSS3**, and **Vanilla JavaScript**. This project showcases a clean UI, smooth animations, and a premium landing page experience without using any frontend framework.

## 🌐 Live Demo

**Website:** https://foxxy-sys.github.io/bean-brew-coffee/

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern and premium coffee shop landing page
* Multi-page website (Home & Menu)
* Smooth scroll animations using AOS
* Mobile-friendly navigation
* Interactive menu section
* Gallery section
* Testimonials section
* WhatsApp contact integration
* SEO-friendly HTML structure

## 🚀 Getting Started

Clone this repository:

```bash
git clone https://github.com/foxxy-sys/bean-brew-coffee.git
```

Open the project folder and launch `index.html` in your browser, or use the **Live Server** extension in Visual Studio Code.
